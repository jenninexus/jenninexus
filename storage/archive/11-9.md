### Critical Bugs Fixed: youtube-grid.js + Duplicate JS Includes

**🐛 BUG #1: youtube-grid.js container_id Ignored**

**ROOT CAUSE:** youtube-grid.js `renderSection()` function (line 244) was **ignoring the `container_id` property** from YAML configs!

**What Was Broken:**
- YAML files use `container_id: "jennistyles-playlists"` to specify custom container IDs
- youtube-grid.js constructed container ID from section name only: `${sectionId}-playlists`
- Example: `featured_section` → looked for `featured-playlists` container
- But HTML had `jennistyles-playlists` container → **MISMATCH!** Videos never loaded

**✅ FIX APPLIED (line 245):**
```javascript
// BEFORE: Always used sectionId
const containerId = `${sectionId}-playlists`;

// AFTER: Respects YAML container_id property
const containerId = sectionConfig.container_id || `${sectionId}-playlists`;
```

**Impact:**
- ✅ Fixes jennistyles.php (now respects `container_id: "jennistyles-playlists"`)
- ✅ Benefits ALL 10 YAML files that use `container_id` property

/**
 * Render blog post header (breadcrumbs, title, meta)
 * 
 * @param string $slug The blog post slug
 * @param array $options Optional: custom title, date, author
 */
function renderBlogHeader($slug, $options = []) {
    $post = getBlogPostData($slug);

            'title' => $options['title'] ?? 'Blog Post',

    echo '<li class="breadcrumb-item"><a href="/">Home</a></li>';
    echo '<li class="breadcrumb-item"><a href="/blog">Blog</a></li>';
    echo '<li class="breadcrumb-item active" aria-current="page">' . htmlspecialchars($post['title']) . '</li>';
    echo '</ol>';
    echo '</nav>';
    
    // Tags
    if (!empty($post['tags'])) {
        foreach ($post['tags'] as $tag) {
            $displayName = ucwords(str_replace('-', ' ', $tag));
            echo '<button type="button" class="badge bg-secondary me-1 tag-badge" ';
            echo 'data-tag-slug="' . htmlspecialchars($tag) . '" ';
            echo 'onclick="(window.tagFilter && window.tagFilter.toggle) ? window.tagFilter.toggle(\'' . htmlspecialchars($tag) . '\') : null">';
            echo htmlspecialchars($displayName);
            echo '</button> ';
        }
    }
    
    // Date and author
    echo '<span class="text-muted ms-2">' . htmlspecialchars($post['date']) . '</span>';
    if (isset($post['author'])) {
        echo '<span class="text-muted ms-2">by ' . htmlspecialchars($post['author']) . '</span>';
    }
    echo '</div>';
    
    // Title
    echo '<h1 class="display-4 mb-4">' . htmlspecialchars($post['title']) . '</h1>';

        echo '<p class="lead text-muted">' . htmlspecialchars($post['excerpt']) . '</p>';
    }
    
    echo '<hr>';
    echo '</header>';
}
