Notes about permissions and execution:
- `~/.ssh` directories must be 700 (drwx------). `authorized_keys` files must be 600 (-rw-------). These are ownership/permission requirements for OpenSSH to accept key authentication.
- The `authorized_keys` file itself should NOT be executable. The execute (`x`) bit is irrelevant to SSH public-key authentication and must not be set on `authorized_keys`.
- Scripts (for example `scripts/fix-permissions-jenninexus-remote.sh`) must be executable to run directly. Ensure those script files (in `/var/www/jenninexus/scripts/`) have the executable bit set (chmod +x) and appropriate ownership (root or sysadmin depending on execution context).
