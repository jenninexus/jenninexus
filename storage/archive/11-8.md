**Root Cause:**
```css
/* BEFORE (custom.css line 138): */
@media (max-width: 575.98px) {
  .btn svg, .btn i.bi, .btn .bi-icon {
    width: 1.25em; /* Too large for btn-sm buttons */

  }
}
```

- `btn-sm`: Icons should be 0.875em (from theme-demo.php line 478)
