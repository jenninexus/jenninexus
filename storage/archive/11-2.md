### JavaScript Event Flow

```
User Clicks Tag Badge
  ↓
Event Listener: document.querySelectorAll('.tag-filter-badge')
  ↓
Read: this.dataset.tagName, this.dataset.tagSlug
  ↓
Toggle: selectedTags.add(tagName) or selectedTags.delete(tagName)
  ↓
Update Badge: classList.add('bg-primary') or classList.add('bg-secondary')
  ↓
Build URL: /tags.php?filters=Tag1,Tag2,Tag3
  ↓
Navigate: window.location.href = newUrl
  ↓
Page Reloads → PHP filters content → Renders new results
```
