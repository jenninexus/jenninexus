## 📝 Key Insights

### Video Grid System
- **Three Layout Patterns**:
  1. Single-Card (gaming.php, gamedev.php, youtube.php) - Interactive grid of playlist cards
  2. Multi-Video (diy.php featured) - Full-width panels showing 3 videos per playlist
  3. Vertical Video (gaming.php shorts) - Portrait 9:16 aspect ratio for YouTube Shorts

- **CSS Architecture**:
  - Bootstrap 5.3.8 provides foundation (grid, aspect ratios)
  - custom.css handles global behaviors (hover effects, play overlays)
  - Theme CSS adds page-specific colors/shadows (gaming-theme.css, gamedev-theme.css, diy-theme.css)
  - youtube-grid.js generates dynamic HTML with proper classes

### Button Usage Patterns
- **Outline Buttons (~80% of all buttons)**: Clean, modern, transparent background, works perfectly with dark mode
  - Used for: CTAs, navigation, platform links, filters, blog "Read More", footer links
- **Platform Buttons (~10-15%)**: YouTube Subscribe, Discord Join, Steam links, Patreon CTAs
- **Social Icons (~5%)**: Square 64-80px responsive icon-only buttons (index.php hero)
- **Specialized (~5%)**: DIY pill buttons, occasional solid buttons for critical actions
