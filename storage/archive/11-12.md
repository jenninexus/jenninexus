**Root Cause**: Bootstrap default carousel controls positioned at `left: 0` and `right: 0`
**Impact**: Visual clutter, poor UX on desktop

**Solution Options**:
1. **Option A: Move controls outside container (RECOMMENDED)**
   ```css
   @media (min-width: 992px) {
     #featuredCarousel .carousel-control-prev {
       left: -50px;
     }
     #featuredCarousel .carousel-control-next {
       right: -50px;
     }
   }
   ```

   ```css
   @media (max-width: 991px) {
   ```
