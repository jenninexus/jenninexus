## 🎨 Design Consistency Notes

### Current SVG System (Verified)
- **Sizing Classes**: .svg-xs (0.75em), .svg-sm (0.875em), .svg-md (1.125em), .svg-lg (1.25em), .svg-xl (1.5em), .svg-2xl (2em), .svg-3xl (3em)
- **Multipliers**: .svg-1x through .svg-5x (1em-5em)
- **Animations**: .svg-spin, .svg-pulse, .svg-beat, .svg-fade, .svg-bounce, .svg-shake
- **Accessibility**: All animations respect prefers-reduced-motion
- **Color**: SVGs use currentColor or CSS custom properties for theme awareness

### Current Theme System (Verified)
- **Toggle Method**: theme-toggle.js toggles `data-bs-theme="light|dark"` on `<html>`
- **Storage**: localStorage.setItem('theme', 'light|dark')
- **Icon**: Inline SVG (moon/sun from FontAwesome 6.7.2) with proper color handling
- **CSS Variables**: Defined in all-themes.css (:root[data-bs-theme="light|dark"])
- **Bootstrap Integration**: Uses --bs-body-bg, --bs-body-color, --bs-primary, etc.
